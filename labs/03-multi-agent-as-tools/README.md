# Lab 03: Multi-agent as tools

## Objectives

- Understand the module outcome.
- Complete the guided build with Azure AI Foundry.
- Validate behavior before moving to the next module.

## Steps

1. Open this lab in VS Code Insiders.
1. Configure `.env` from `/shared/.env.example`.
1. Complete implementation in `/src`.
1. Compare against `/solution` if needed.

## Validation

- Run the validation command documented for this module.
- Confirm expected outputs and successful tool interactions.

## Troubleshooting

- Verify Azure login (`az login`) and subscription context.
- Confirm access to the assigned Foundry project.
- Re-run shared health checks from `/scripts/health-check.py`.
