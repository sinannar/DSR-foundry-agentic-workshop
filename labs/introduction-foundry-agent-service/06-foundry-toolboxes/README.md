# Step 06: Foundry Toolboxes (optional)

## Objectives

- Understand what a Foundry Toolbox provides to an agent.
- Attach a toolbox to the agent and demonstrate its capabilities.

## Steps

1. Open the `retail-assistant` agent.
1. Review the available Foundry Toolboxes and what each bundles together.
1. Attach a toolbox to the agent.
1. Update the agent instructions to describe when to use the toolbox.
1. Run the agent with a prompt that exercises a toolbox capability.
1. Inspect the run and confirm the toolbox capability was used.

## Validation

- The toolbox appears in the agent configuration.
- A test prompt exercises a toolbox capability and the response reflects it.
- The run trace shows the toolbox capability being used.

## Troubleshooting

- If the toolbox is unavailable, skip this optional step; it is not required
  for later steps.
- If a capability is never used, strengthen the agent instructions and use a
  prompt that clearly requires it.
- If access fails, confirm your role grants access to the project.
